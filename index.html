My Portfolio

Visit Sinai

This is a static-site template that I built using HTML, CSS, JavaScript and the Materialize styling framework. It is a mock tourism site for Sinai Egypt. This was a homework assignment that I decided to build out on my own.

What's For Dinner?

This is a full-stack application using MongoDB, Express, React, and Node.js that allows users to create a personal account, create recipes, and favorite recipes from an existing database. This app was made as group project at General Assembly. We were given 5 days to complete the project.

Book Club

This is a site I created using Ruby on Rails for the Back-end and React for the Front-end. You can add or delete from a list of authors and add a photo of a book and author as well. Plans for future development include user authentication, linking to an API for book reviews, and a discussion forum, and re-working the styling. This was a solo project, which we had 5 days to complete.

Travel Site

This is a full-stack application using Ruby on Rails. In this app a user can post a review about a city they have visited. The site features full CRUD on the posts which is tied to each city. This project was created in 3 days with a group of 3 people at General Assembly.
