My Portfolio

Visit Sinai

This is a static-site template that I built using the Materialize styling framework. Materilize has great buttons and icons which add to the user experience. I also like the color pallet they offer, I think it really complimented the photos that I decided to use. This is a mock tourism site for Southern Sinai Egypt, a place that I used to work and live.

What's For Dinner?

This is a full-stack application using Node/Express and Mongo/Mongoose. It allows users to create a personal account, create recipes and favorite recipes from an existing database. The app was styled using CSS and Materialize. This was a group project at General Assembly, we were given 5 days to complete the project.

Conscious Path

Before I started doing web development, I worked in hospitality and luxury spas. Over the years, I have become familiar with different types of healing therapies. This is a mock site for a wellness center and it's what I would imagine as the perfect spa. It was built using HTML, CSS, JavaScript and Bootstrap.

MindHak

This is a Wordpress site that was completed in two days at the GoodieNation "Hack The Violence" Hackathon. It helps people learn about emotional intelligence, mindfulness, and self-compassion. The site also uses an IBM Watson chatbot that responds to user's emotions with tips and resources on how to become more mindful.

Book Club

This is a React on Rails app. I used React for the frontend and Rails for the backend. You can add or delete from a list of authors and add a photo of an author as well. Plans for future development include user authentication, linking to an API for book reviews, a discussion forum and reworking the styling. This was a solo project which I had 5 days to complete.

InVision Prototype

This is a prototype I built using InVision. The photo assets were provided to me by General Assembly. I look forward to working with the new InVision Studio when it launches in 2018.
